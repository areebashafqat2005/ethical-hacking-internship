# Week 3

## Manual & Automated Enumeration After Initial Access

This week focused on performing detailed manual and automated enumeration of the compromised Metasploitable 2 system after gaining initial access.

### Topics Covered
- System Information Enumeration
- User and Group Enumeration
- Privilege Enumeration
- Network Enumeration
- File System Enumeration
- Sensitive File and Directory Identification
- Automated Linux Enumeration
- Manual vs Automated Findings Comparison
- Privilege Escalation Checks
- Evidence Collection and Reporting

### Tools Used
- Kali Linux
- Metasploitable 2
- LinEnum
- Linux Command-Line Utilities
- Netstat
- Metasploit Framework
